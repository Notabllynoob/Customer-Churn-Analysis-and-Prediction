# Customer-Churn-Analysis-and-Prediction
This project provides a full-stack solution for predicting customer churn in the telecommunications industry. Moving beyond a static notebook, it transforms raw data into a live, interactive web application, demonstrating an end-to-end workflow from data analysis to a deployed production-ready model.
